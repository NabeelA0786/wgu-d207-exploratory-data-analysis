# D207 – Exploratory Data Analysis

**WGU Master of Science in Data Analytics**

## Overview
Performed exploratory data analysis on a cleaned telecom churn dataset to uncover patterns, relationships, and business insights driving customer attrition.

## Dataset
Telecom customer churn dataset (cleaned output from D206)

## What Was Done
- Examined customer demographics and service usage patterns
- Built visualizations: histograms, scatterplots, boxplots, and correlation heatmaps
- Identified key statistical relationships between features and churn behavior
- Generated actionable business recommendations based on findings

## Tools & Libraries
`Python` · `Pandas` · `Matplotlib` · `Seaborn` · `Jupyter Notebook`
